# Subtitle Optimizer

A small Windows desktop tool for German learners. It cleans fragmented SRT subtitles, especially subtitles generated by CapCut, and turns them into more complete sentence-based subtitles.

## Features v1.0

- Drag and drop `.srt` files into the app
- Select one or multiple SRT files manually
- Merge subtitle fragments into complete German sentences
- Preserve the original timeline by using the first fragment start time and the last fragment end time
- Limit long subtitles by character count
- Generate `*_clean.srt` next to the original file
- Fully offline

## Download Windows EXE from GitHub Actions

1. Upload this project to your GitHub repository.
2. Open the **Actions** tab.
3. Click the latest **Build Windows EXE** run.
4. Download the artifact named **SubtitleOptimizer-Windows**.
5. Unzip it and run `SubtitleOptimizer.exe`.

## Local development

```bash
pip install -r requirements.txt
python src/subtitle_optimizer.py
```

Build locally on Windows:

```bash
pip install -r requirements.txt
pip install pyinstaller
pyinstaller --noconfirm --clean --onefile --windowed --name SubtitleOptimizer src/subtitle_optimizer.py
```

The EXE will be created in `dist/SubtitleOptimizer.exe`.

## Recommended workflow

```text
Tagesschau video
  ↓
CapCut auto captions
  ↓
Export SRT
  ↓
Subtitle Optimizer
  ↓
PotPlayer with *_clean.srt
```

## Notes

This v1.0 uses rule-based processing. It does not use AI and does not send files online.
